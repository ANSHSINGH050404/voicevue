# voicevue

VoiceVue — AI voice interview platform with Next.js and Drizzle ORM.

## Stack

Next.js, Drizzle, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/voicevue.git
cd voicevue
npm install
npm run dev   # or: npm start
```
